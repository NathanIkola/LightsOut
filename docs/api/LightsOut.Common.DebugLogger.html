Class DebugLogger

Handles all of the debug logging needs for my mod. Respects the mod settings option for printing extra debug messages. Adds a header to the beginning of all messages passed in so they are easy to identify in the log.

Assert(Boolean, String, Boolean)

Used to catch places where an expression is expected to be true, but is actually false. Useful for finding places where code is not behaving correctly. Does not log unless an error is found.

Log this assertion failure only one time
Remarks

If onlyOnce is set, this uses the hash of failMessage to only log the message once. If you make failMessage have dynamic content, it will appear once for every unique version of failMessage that is passed in.
